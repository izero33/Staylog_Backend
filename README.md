# Staylog - 숙소 예약 플랫폼(Backend)
팀 프로젝트로 진행한 숙소예약 서비스 Staylog의 백엔드 저장소입니다. 
Spring Boot 기반의 REST API 서버를 구현했습니다.

# 주요기능
1) 숙소 검색
- 지역코드 기반 숙소 조회
- 메인화면 최신순/별점순/리뷰많은 순/낮은가격순 정렬
3) 숙소, 객실 상세 페이지
- 숙소, 객실별 상세 조회
- 예약 불가일 조회
4) 예약 기능
- 체크인, 체크아웃 날짜 검증
- 총 숙박 금액계산
5) 게시판
- 리뷰게시판
- 인증받은 사용자만 작성 가능한 저널 게시판
6) 관리자 가능
- 숙소 등록/수정
- 회원관리
- 게시물 관리

#역할
- 메인페이지
  - 최신순 / 별점순 / 리뷰순 / 낮은가격순 정렬
  - Oracle 기반 가격·리뷰 데이터 조합 및 조회 최적화
- 예약 불가일(Blocked Dates) 로직 구현
  - 예약 중복 검증
  - 3개월 조회 제한 처리
- Kakao Maps API 연동 기능 개발
  - 주소 → 위도/경도 변환
- 객실 상세 페이지 데이터 처리
  - 객실 정보 및 날짜별 예약 가능 여부 조회
  - 이미지·가격·옵션 데이터 제공
